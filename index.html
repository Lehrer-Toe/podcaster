⚠️ Mikrofon-Zugriff erforderlich

Ihr Gerät erlaubt den Zugriff auf das Mikrofon nicht. So können Sie den Zugriff aktivieren:

  • Stellen Sie sicher, dass Sie "Erlauben" drücken, wenn nach Mikrofonzugriff gefragt wird
  • Überprüfen Sie die Einstellungen Ihres Browsers oder Geräts, um den Mikrofonzugriff zu aktivieren
  • Auf iOS: Öffnen Sie die Einstellungen > Safari > Seitenberechtigung > Mikrofon und aktivieren Sie den Zugriff

📱 Hinweis für iOS-Nutzer

Diese App verwendet automatisch ein mit Ihrem Gerät kompatibles Audioformat. Es ist möglich, dass die Aufnahmequalität auf iOS-Geräten etwas geringer ist als auf Desktop-Browsern.

Vorhandene Aufnahmen

    Verwendung

    1. Geben Sie einen Dateinamen ein
    2. Klicken Sie auf "Aufnahme starten"
    3. Sie können die Aufnahme pausieren und fortsetzen
    4. Nach dem Stoppen können Sie die Aufnahme anhören
    5. Klicken Sie auf "Hochladen", um die Aufnahme zu speichern

    Hinweis: Ihr Browser speichert die letzte Aufnahme lokal. Sie können die Seite neu laden, ohne die Aufnahme zu verlieren.

    Fehlerbehebung

    • Mikrofon funktioniert nicht: Prüfen Sie die Browsereinstellungen und erteilen Sie die Berechtigung für den Mikrofonzugriff.
    • Aufnahme startet nicht: Stellen Sie sicher, dass ein Dateiname eingegeben wurde.
    • Probleme auf iOS: Starten Sie Safari neu oder verwenden Sie den Privatmodus, falls Sie Probleme haben.
    • Hochladen funktioniert nicht: Möglicherweise ist die Verbindung zum Server unterbrochen. Versuchen Sie es später erneut.
